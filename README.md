# Whitespace Linter

> Basic linter that detects and formats trailing whitespace and extra empty lines.

## Features

- Detects excessive whitespace in your file.
- Allows you to fix detected errors.

## Built With

- Ruby
- Rspec

- Ruby

## Install

```bash
gem install ws_lint
```

## Usage

```bash
ws_lint /path/to/file
```

## Testing

```bash
bundle exec rspec
```
